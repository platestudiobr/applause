Estados de Interação
😴
Repouso
Personagem aguarda interação ou próxima ação. Animação em loop suave.
⏳
Transição
Entre estados. Fade in/out, mudança de posição, preparação para ação seguinte.
🎬
Execução
Animação em plena ação. Efeitos visuais, particles, transformações, movimento dinâmico.
✨
Conclusão
Resultado visual. Sucesso, falha ou mudança de contexto. Volta ao repouso ou encadeia próxima.
Padrões de Interação
📝 Chat → Trigger → Animação
Usuário escreve no chat que completou meta. Sistema detecta palavra-chave. Animação no canvas responde com celebração.
Usuário: "Vendi 1000" ↓ Backend detecta: "vendi" + número ↓ Emit: { type: 'sold', amount: 1000 } ↓ Canvas: celebração + contador
🖱️ Click → Elemento → Sequência
Usuário clica em elemento interativo (caixa, baú, presente). Animação responde. Próxima ação dispara automaticamente.
Usuário clica em 📦 ↓ Event listener: onClick() ↓ Play: 'open_box' animation ↓ Emit: nova missão no chat ↓ Auto-play: próxima animação
⏰ Auto-Trigger → Sequência Encadeada
Primeira animação termina, automaticamente dispara a próxima. Cria fluxo visual contínuo e narrativo.
Anim 1 termina (2s) ↓ onComplete callback ↓ Pause(1s) ↓ Play: Anim 2 ↓ Chat mostra conteúdo
🔔 Notificação → Atenção Visual
Sistema avisa novo evento. Personagem reage subtilmente para chamar atenção sem ser intrusivo.
Sistema: "nova missão" ↓ Anim: pulse / sway leve ↓ Chat: aparece mensagem ↓ Usuário pode ignorar ou clicar
